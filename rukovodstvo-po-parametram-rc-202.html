Настройки Input FX / Track FX

Основная процедура редактирования эффекта

1  

Нажмите кнопку [P. SHIFT] - кнопка [DELAY], чтобы она начала мигать.

2  

Поверните регулятор [VALUE], чтобы выбрать тип FX.

3  

Нажмите ручку [VALUE], чтобы выбрать параметр; затем поверните ручку [VALUE], чтобы отрегулировать значение.

4  

Нажмите кнопку [SETUP] (EXIT), чтобы вернуться к экрану воспроизведения.

Увеличьте это значение, если шума больше, и уменьшите его, если шума меньше.
0–100 (по умолчанию: 40)

Регулирует глубину шумоподавителя.

Синхронизация темпа (MIDI Sync)

Вы можете синхронизировать темп с данными синхронизации MIDI, полученными через разъем MIDI IN или порт USB.
Вы также можете использовать MIDI-кабель для синхронизации двух устройств RC-202.

Ссылка

Более подробную информацию о MIDI и синхронизации см. В разделе «Настройки MIDI» (Руководство пользователя).

В

 (дефолт)

RC-202 обычно работает со своим внутренним темпом, но синхронизирует темп с MIDI Clock, если MIDI 

данные часов вводятся через разъем MIDI IN или порт USB (AUTO).
Выберите настройку «АВТО», если RC-202 используется в качестве ведомого устройства.

INT

RC-202 будет работать в темпе памяти фраз, заданном на устройстве (INTERNAL).
Выберите настройку «ВНУТРЕННИЙ», если вы не хотите синхронизировать RC-202 с внешним устройством.

SRC

Выбор MIDI или USB для синхронизации (источник синхронизации MIDI)

Определяет, будет ли RC-202 синхронизироваться с данными темпа из порта USB или данными темпа из разъема MIDI IN, когда MIDI Sync 

"АВТО."

В

 (дефолт)

При подключении через USB RC-202 будет синхронизироваться с данными темпа из порта USB. Когда не подключен 

через USB, он будет синхронизироваться с данными темпа из разъема MIDI IN.

MID

RC-202 будет синхронизироваться с темпом через разъем MIDI IN.

RX

Канал приема MIDI

1–16 (по умолчанию: 1)

OMn

ВЫКЛ.

НА

 (дефолт)

Сообщения принимаются по всем MIDI-каналам, независимо от настроек MIDI-канала приема.

TX

1–16

Устанавливает MIDI-канал, используемый для передачи MIDI-сообщений.

Rx

 (дефолт)

Канал передачи MIDI будет таким же, как канал приема MIDI.

PC

Приложение

О папке «WAVE» RC-202

Если вы хотите загрузить отдельные файлы WAV с вашего компьютера во фразовую память RC-202, подключите RC-202 к вашему компьютеру и скопируйте 

файлы в папку «ROLAND» - «WAVE» накопителя BOSS_RC-202.

Ссылка

Для получения подробной информации о том, как загружать файлы в RC-202, обратитесь к разделу «Использование USB для обмена файлами с вашим компьютером (USB Mass Storage)» (Руководство пользователя).
